# 校园梦想家 - UI设计规格说明

## 1. 设计愿景

打造一款具有现代科技感和沉浸式体验的校园模拟经营游戏UI。设计风格融合赛博朋克的霓虹美学与传统校园元素，营造出既有未来感又亲切温暖的视觉效果。

## 2. 设计语言

### 美学方向
- 主风格：深色赛博朋克 + 霓虹光效
- 辅助风格：毛玻璃质感 + 渐变流光
- 特色元素：粒子动画 + 光影变幻

### 配色方案
- **主色调**: #6366f1 (靛蓝紫) - 品牌色
- **次要色**: #10b981 (翠绿) - 成功/健康
- **强调色**: #f59e0b (琥珀橙) - 警告/能量
- **危险色**: #ef4444 (红色) - 错误/危险
- **背景深色**: #0f172a (深蓝黑)
- **背景渐变**: #1e1b4b → #312e81
- **文字主色**: #ffffff
- **文字次色**: #94a3b8

### 字体选择
- 主字体: Noto Sans SC (中文支持优秀)
- 备选: -apple-system, BlinkMacSystemFont

### 空间系统
- 基础单位: 4px
- 卡片圆角: 24px - 32px
- 按钮圆角: 12px - 16px
- 内边距: 16px - 32px

### 动效哲学
- 入场动画: 弹性缓动，duration 400-600ms
- 悬停效果: 平滑过渡，duration 200-300ms
- 背景动画: 缓慢呼吸，duration 15-20s循环
- 粒子效果: 自然漂浮，增强沉浸感

### 视觉资源
- 图标: Emoji符号 (高兼容性)
- 装饰: CSS渐变 + SVG几何图形
- 粒子: Canvas绘制光点

## 3. 布局与结构

### 页面层次
1. **启动页** - 全屏沉浸式登录界面
   - 动态粒子星空背景
   - 浮动光球装饰
   - 中心LOGO展示
   - 菜单按钮组

2. **创建角色页** - 分步式表单
   - 顶部返回导航
   - 卡片式选项
   - 平滑切换动画

3. **游戏主页** - 仪表盘布局
   - 顶部状态栏 (玩家信息 + 时间 + 速度控制)
   - 左侧导航栏 (桌面) / 底部导航栏 (移动)
   - 主内容区 (动态页面切换)

## 4. 功能与交互

### 启动界面
- 粒子背景跟随鼠标微微移动
- LOGO持续浮动动画
- 按钮悬停发光扩散
- 版本信息呼吸效果

### 角色创建
- 角色卡片3D翻转效果
- 选项卡片边框流光
- 确认按钮脉冲光晕
- 表单输入聚焦高亮

### 游戏界面
- 导航按钮点击涟漪效果
- 卡片悬停上浮+阴影加深
- 进度条流光动画
- 状态数值变化动画

### 错误与反馈
- Toast通知滑入动画
- 模态框缩放入场
- 按钮禁用状态灰度处理

## 5. 组件清单

### 按钮组件
- **主按钮**: 渐变背景 + 发光阴影 + 悬停放大
- **次按钮**: 透明背景 + 边框 + 悬停填充
- **图标按钮**: 圆形/圆角 + 图标居中
- **速度按钮**: 小尺寸 + 状态切换高亮

### 卡片组件
- **角色卡片**: 图标 + 标题 + 描述 + 选中高亮
- **仪表盘卡片**: 标题栏 + 内容区 + 悬停效果
- **统计卡片**: 数值展示 + 进度指示

### 表单组件
- **文本输入**: 聚焦边框发光 + 背景加深
- **下拉选择**: 自定义箭头 + 选项样式
- **单选按钮**: 图标 + 选中填充

### 状态指示
- **进度条**: 渐变填充 + 流光动画
- **标签徽章**: 圆角胶囊 + 状态色彩
- **加载动画**: 旋转环/脉冲点

## 6. 技术实现

### 前端技术
- HTML5 + CSS3 + JavaScript ES6+
- 无外部依赖 (单文件部署)
- CSS变量实现主题管理
- CSS Grid + Flexbox布局

### 动画实现
- CSS @keyframes 定义动画
- CSS transition 平滑过渡
- requestAnimationFrame 游戏循环
- Canvas 粒子系统 (可选增强)

### 响应式策略
- 移动优先设计
- 断点: 768px (桌面/移动切换)
- 触摸优化 (点击区域 ≥ 44px)

### 性能优化
- GPU加速 (transform, opacity)
- 减少重绘 (will-change提示)
- 懒加载非首屏资源
- 本地存储缓存游戏数据
